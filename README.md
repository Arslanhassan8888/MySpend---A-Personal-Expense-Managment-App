# MY_SPEND: Personal Expense Management App

## Problem Statement

Many people struggle to monitor their daily spending, especially with digital payments making it easy to overspend. Small purchases add up quickly, and existing apps can be confusing, costly, or unsafe. There is a need for a simple, secure, and accessible tool that helps users understand and manage their finances clearly.

## Proposed Solution

MySpend is a simple and user-friendly web app designed to help users record, organise, and review their expenses. It encourages financial awareness by focusing on reflection rather than automation. The design is clean, accessible, and suitable for users with different levels of digital experience.

## Key Features
- Add, edit, and delete expenses

- Categorise spending for clarity

- View weekly and monthly summaries

- Simple layout suitable for all users

- Visual charts to show spending patterns

- Secure data handling

- Encourages long-term financial learning

## Technology Stack

### Frontend

- **HTML5** → Page structure  

- **CSS3** → Styling and layout 

- **JavaScript** → Interaction and validation  

- **Chart.js** → Visual charts 

### Backend

- **Python** → System logic  

- **Flask** → Routing, sessions, and web framework 

- **Jinja2** → Dynamic templates  

- **Database** → For storing expenses (specify: SQLite/PostgreSQL/MySQL)  

- **Visual Studio Code** → Development environment
